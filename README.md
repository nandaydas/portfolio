# Nanday Das - Portfolio

Welcome to the source code of my personal portfolio website: [nandaydas.in](https://nandaydas.in).

## About Me

I'm **Nanday Das**, a Flutter specialist dedicated to building high-performance, beautiful mobile experiences that run natively on any screen. I bridge the gap between high-end native performance and efficient cross-platform development.

## Project Structure

This portfolio is built with:
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Supabase (Backend/CRUD operations)

## Pages

- `index.html`: The main landing page showcasing my work, skills, and client feedback.
- `pages/contact.html`: The contact form and inquiry page.
- `pages/privacy-policy.html`: Privacy Policy for the website.
- `pages/terms-of-service.html`: Terms of Service for the website.

## Capabilities & Ecosystem

- **Primary Stack:** Flutter & Dart
- **Backend & Cloud:** Firebase, Supabase, Node.js, REST / GraphQL
- **Core Skills:** State Management, Custom Animations, CI/CD Pipelines, Dart Architecture
